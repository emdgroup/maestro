  <action>
Edit the `app.security.csp` string in `src-tauri/tauri.conf.json` to add three allowances:

Current CSP:
```
default-src 'self'; script-src 'self'; style-src 'self' 'unsafe-inline'; img-src 'self' data:; connect-src 'self' https://api.github.com https://*.atlassian.net; font-src 'self' data:
```

Updated CSP (changes marked):
```
default-src 'self'; script-src 'self'; style-src 'self' 'unsafe-inline' https://fonts.googleapis.com; img-src 'self' data:; connect-src 'self' ipc: http://ipc.localhost https://api.github.com https://*.atlassian.net; font-src 'self' data: https://fonts.gstatic.com
```

Three changes:
1. `style-src`: append `https://fonts.googleapis.com` — allows loading the Google Fonts CSS stylesheet (the `@import url(...)` in `src/index.css`)
2. `connect-src`: insert `ipc: http://ipc.localhost` after `'self'` — allows Tauri's IPC custom protocol so it does not fall back to postMessage. The `ipc:` scheme handles the custom protocol and `http://ipc.localhost` handles the HTTP endpoint variant (e.g., `plugin:window|show`).
3. `font-src`: append `https://fonts.gstatic.com` — allows downloading the actual .woff2 font files served by Google Fonts CDN

Do NOT change any other directives or values. Keep the single-line CSP string format.
