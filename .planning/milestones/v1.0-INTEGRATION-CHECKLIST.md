   ├→ spawn_agent_cli_pty (Phase 5)
   └→ attach_terminal (Phase 5)
   ↓
5. Terminal streams output (Phase 5)
   ↓
6. User moves task to Review (Phase 2)
   ↓
7. TaskCard Review button
   ↓
8. ReviewModal opens → get_diff_for_review (Phase 6)
   ├→ Dispatcher (Phase 9 for remote)
   └→ DiffViewer displays
   ↓
9. ApprovalForm Approve → approve_task_and_merge (Phase 6)
   ↓
10. Handler spawns merge (sidecar --merge)
    ↓

    ├→ Task status → Done
    ├→ Worktree status → Dirty

    ↓

    ├→ git worktree remove --force
    ├→ git branch -D
    └→ git worktree prune
