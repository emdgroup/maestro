Create `src/store/navigationStore.ts` with the following exact structure:

**Exported types:**
```typescript
export type ViewType = "kanban" | "agents" | "worktrees" | "settings";
export type SubView = "backlog" | "board" | "archive";
export type NavigationTarget =
  | { taskId: string }
  | { agentId: string }
  | { worktreeId: string }
  | { view: "backlog" | "board" | "archive" | "agents" | "worktree" | "settings" };
```

**Internal constants:**
```typescript
const PAGE_ORDER: Record<ViewType, number> = { kanban: 0, agents: 1, worktrees: 2, settings: 3 };
```

**Helper function:**
```typescript
function targetViewToTab(view: string): ViewType {
  if (view === "worktree") return "worktrees";
  if (view === "backlog" || view === "board" || view === "archive") return "kanban";
  return view as ViewType;
}
```

**Store interface — NavigationState:**
- `activeTab: ViewType` (default: "kanban")
- `slideDirection: number` (default: 1)
- `activeSubView: SubView` (default: "board")
- `pendingTaskId: string | null` (default: null)
- `pendingAgentId: string | null` (default: null)
- `pendingWorktreeId: string | null` (default: null)
- `navigate: (target: NavigationTarget) => void`
- `setActiveTab: (tab: ViewType) => void`
- `setActiveSubView: (sub: SubView) => void`
- `clearPendingTask: () => void`
- `clearPendingAgent: () => void`
- `clearPendingWorktree: () => void`

**navigate() logic:** Use `'taskId' in target` / `'agentId' in target` / `'worktreeId' in target` / `'view' in target` branching. Compute slideDirection INSIDE the `set()` callback using `PAGE_ORDER[newTab] > PAGE_ORDER[state.activeTab] ? 1 : -1`. For `setActiveTab`, guard with `if (tab !== state.activeTab)` before updating direction.

**Selector hooks (all exported):**
- `useActiveTab`, `useSlideDirection`, `useActiveSubView`
- `usePendingTaskId`, `usePendingAgentId`, `usePendingWorktreeId`
- `useNavigate` (returns `navigate` function)
- `useNavigationActions` (returns object with `setActiveTab`, `setActiveSubView`, `clearPendingTask`, `clearPendingAgent`, `clearPendingWorktree`)

Follow the exact Zustand + Immer pattern from `src/store/configStore.ts`: `create<NavigationState>()(immer((set) => ({ ... })))`.
