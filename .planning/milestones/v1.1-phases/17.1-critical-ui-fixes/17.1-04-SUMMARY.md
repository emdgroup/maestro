## Decisions Made

- **Chromium as default browser** - Firefox and WebKit commented out due to system library dependencies (libevent-2.1.so.7). Easy to re-enable in full environments.
- **Headless-first test design** - Used addInitScript for dark mode instead of page.evaluate to avoid localStorage security errors
- **domcontentloaded instead of networkidle** - Faster, more reliable in test environments; capture happens after interactive content is ready
- **Root element as primary selector** - #root always available even when app is loading; avoids "element not found" errors on conditional rendering
- **Baseline-on-first-run pattern** - Tests fail with "snapshot doesn't exist" on first run (creating baseline), then pass on subsequent runs
- **Error filtering** - Expected Tauri/debug logs excluded from error detection to focus on genuine rendering issues
