<task type="auto">
  <name>Task 1: Initialize shadcn/ui and generate components.json</name>
  <files>components.json</files>
  <action>
    Initialize shadcn/ui configuration:
    1. Run: `pnpm dlx shadcn-ui@latest init -y --cwd .`
       This generates components.json with defaults
    2. Update components.json manually to customize:
       ```json
       {
         "$schema": "https://ui.shadcn.com/schema.json",
         "style": "default",
         "rsc": false,
         "tsx": true,
         "baseColor": "neutral",
         "cssVariables": true,
         "prefix": "",
         "aliases": {
           "@/components": "./src/components",
           "@/ui": "./src/components/ui",
           "@/lib": "./src/lib",
           "@/hooks": "./src/hooks"
         }
       }
       ```
    3. Ensure CSS variable mode is enabled (`"cssVariables": true`) — allows system accent color integration
    4. Set baseColor to "neutral" (uses neutral grays, accent via CSS variables)
    5. Verify tsconfig.json has matching path aliases for @/components and @/ui
    6. Run: `pnpm install` to ensure all dependencies updated
  </action>
  <verify>
    cat components.json | grep -q '"cssVariables": true' && echo "✓ shadcn/ui configured"
  </verify>
  <done>
    components.json exists with shadcn/ui configuration, CSS variables enabled, correct aliases
  </done>
