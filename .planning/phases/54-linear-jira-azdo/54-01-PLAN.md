- TeamsConnection { nodes: Vec<LinearTeam> } — #[derive(serde::Deserialize)]
- LinearTeam { id: String, name: String, key: String } — #[derive(Debug, Clone, serde::Serialize, serde::Deserialize, specta::Type)] + #[specta(export)] (per D-04, RESEARCH.md Pattern 5)

QUERY CONSTANTS (per PATTERNS.md):
- VIEWER_QUERY: &str = "{ viewer { id name } }"
- TEAMS_QUERY: &str = "{ teams { nodes { id name key } } }"
- ISSUES_QUERY_ALL: &str = r#"{ issues(first: 100) { nodes { identifier title description url updatedAt labels { nodes { name } } } } }"#
- ISSUES_QUERY_TEAM: &str = r#"query IssuesByTeam($teamId: ID!) { issues(filter: { team: { id: { eq: $teamId } } }, first: 100) { nodes { identifier title description url updatedAt labels { nodes { name } } } } }"#

HELPER: post_graphql_query(client: &reqwest::Client, token: &str, query: &str, variables: serde_json::Value) -> Result<reqwest::Response, String> — builds and sends a POST to "https://api.linear.app/graphql" with Authorization: Bearer {token} header and Content-Type: application/json. Returns Err("Linear: Network error: ...") on send failure.

FUNCTION: validate_and_store(project_id: i32, api_key: &str, project_path: &str, app_state: &crate::db::AppState) -> Result<String, String>
- Builds reqwest::Client with 15s timeout
- Posts VIEWER_QUERY with variables: serde_json::Value::Null
- Checks HTTP status: 401 → "Linear: bad credentials", other non-2xx → "Linear API error {status}: {reason}"
- Deserializes body as graphql_client::Response<ViewerResponseData>
- If gql.errors is Some and non-empty → return Err("Linear: {errors[0].message}")
- Extracts display_name = gql.data.ok_or("Linear: empty response")?.viewer.name
- Saves TicketingConfig { provider: Some(ProviderConfig::Linear(LinearConfig { team_id: None })), updated_at: now_rfc3339() } via config.save_to_project(project_path)?
- Stores StoredToken { access_token: api_key.to_string(), refresh_token: None, expires_at: None, provider: "linear".to_string() } via app_state.token_manager.store_token(...)
- Returns Ok(display_name)

FUNCTION: list_teams(token: &str) -> Result<Vec<LinearTeam>, String>
- Builds reqwest::Client with 15s timeout
- Posts TEAMS_QUERY
- Deserializes as graphql_client::Response<TeamsResponseData>
- Returns gql.data.ok_or("Linear: empty response")?.teams.nodes

FUNCTION: fetch_issues(token: &str, team_id: Option<&str>) -> Result<Vec<RemoteIssue>, String>
- Builds reqwest::Client with 15s timeout
- If team_id.is_none(): posts ISSUES_QUERY_ALL with variables: serde_json::Value::Null
- If team_id.is_some(): posts ISSUES_QUERY_TEAM with variables: serde_json::json!({ "teamId": team_id.unwrap() })
- Deserializes as graphql_client::Response<IssuesResponseData>
- Maps each node to RemoteIssue:
  - external_id: format!("linear:{}", issue.identifier)
  - title: issue.title
  - body: issue.description
  - url: issue.url
  - labels: issue.labels.nodes.into_iter().map(|l| l.name).collect()
  - updated_at: issue.updated_at
- Returns Ok(remote_issues)
