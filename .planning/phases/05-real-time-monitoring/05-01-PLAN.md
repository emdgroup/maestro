   - PTY reader task:
     - Read from PTY master in 4096-byte chunks
     - Decode with utf8-lossy to handle mid-sequence UTF-8
     - Send to mpsc channel (backpressure applied by bounded buffer)
     - Break on read error or channel.send error
   - Frontend sender task:
     - Receive from mpsc channel
     - Send to output_channel
     - Break on output_channel.send error (frontend detached)
   - Use `tokio::spawn` for background tasks

2. Implement `send_terminal_input` handler in handlers.rs:
   ```rust
   #[tauri::command]
   async fn send_terminal_input(
       state: tauri::State<'_, Arc<AppState>>,
       task_id: i32,
       input: String,
   ) -> Result<(), String> {
       let sessions = state.pty_sessions.lock().await;
       let session = sessions.get(&task_id)
           .ok_or_else(|| format!("No PTY session for task {}", task_id))?
           .clone();
       drop(sessions);

       let mut session = session.lock().await;
       session.write_input(input.as_bytes()).await
   }
   ```

       }
   }
