`try_gh_cli_token() -> Option<String>`: use `which::which("gh")` check, then `TokioCommand::new("gh").args(["auth", "token"]).output().await.ok()?`, return trimmed stdout or None.

`validate_and_store(project_id: i32, owner: &str, repo: &str, token: Option<String>, project_path: &str, app_state: &crate::db::AppState) -> Result<String, String>`:
1. Resolve token: if None, call `try_gh_cli_token().await`; if still None, return `Err("GitHub: gh CLI not available or not authenticated. Provide a PAT.")`
2. Build reqwest client with 15s timeout
3. GET `https://api.github.com/user` with `Authorization: Bearer {token}` + `User-Agent: maestro/1.0`; check status; deserialize GitHubUserResponse
4. Build `TicketingConfig { provider: Some(ProviderConfig::Github(GitHubConfig { owner: owner.to_string(), repo: repo.to_string() })), updated_at: now_rfc3339() }` and call `config.save_to_project(project_path)?`
5. Store `StoredToken { access_token: token, refresh_token: None, expires_at: None, provider: "github".to_string() }` via `app_state.token_manager.store_token(project_id, stored_token, &app_state.app_data_dir, &app_state.app_handle)?`
6. Return `Ok(user.login)`

`fetch_issues(owner: &str, repo: &str, token: &str) -> Result<Vec<RemoteIssue>, String>`:
1. Build reqwest client
2. GET `https://api.github.com/repos/{owner}/{repo}/issues?state=open&per_page=100`
3. Filter: keep only items where `pull_request.is_none()`
4. Map to RemoteIssue with `external_id: format!("github:{}", issue.number)`, labels from `issue.labels.iter().map(|l| l.name.clone()).collect()`

Use `crate::models::project_config::now_rfc3339` for timestamp.

**fetch_remote_issues:**
```rust
pub async fn fetch_remote_issues(
    app_state: State<'_, Arc<AppState>>,
    project_id: i32,
) -> Result<Vec<RemoteIssue>, String> {
    let project_path = {
        let conn = app_state.db.lock().map_err(|e| format!("Lock failed: {}", e))?;
        conn.query_row("SELECT path FROM projects WHERE id = ?", [project_id],
            |row| row.get::<_, String>(0))
        .map_err(|_| format!("Project {} not found", project_id))?
    };
    let config = TicketingConfig::load_from_project(&project_path)
        .map_err(|_| "No ticketing provider configured".to_string())?;
    let provider = config.provider
        .ok_or_else(|| "No ticketing provider configured".to_string())?;
    let token = app_state.token_manager
        .get_token(project_id, &app_state.app_data_dir, &app_state.app_handle)?
        .ok_or_else(|| "No stored credentials found".to_string())?;
    match provider {
        ProviderConfig::Github(cfg) =>
            crate::ticketing::github::fetch_issues(&cfg.owner, &cfg.repo, &token.access_token).await,
        ProviderConfig::Gitlab(cfg) =>
            crate::ticketing::gitlab::fetch_issues(&cfg.instance_url, cfg.project_id, &token.access_token).await,
        ProviderConfig::Forgejo(cfg) =>
            crate::ticketing::forgejo::fetch_issues(&cfg.instance_url, &cfg.owner, &cfg.repo, &token.access_token).await,
        _ => Err("Provider not yet supported in this phase".to_string()),
    }
}
```
